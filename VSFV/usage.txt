Please place all your files in the VSFV folder
Usage:

bash VSFV.sh \     
  -f contig_test.fa \ # Set your fasta file with unknown contigs
  -b benchmark_test.txt \ # Input your benchmark contig names; remove this line to use self-defined fictitious vsiRNA_simulant files 
  -r sRNA_test.fastq \ # Set your single virus-infected sequencing dataset
  -o ./result_1 \ # Set your output folder
  -m sizeX5ntXstr \ # Set your sRNA feature vector dimension: size, size_P_5nt, sizeXstr, sizeX5nt, sizeX5ntXstr 
  --force \ # Rerun to cover previous results in the same output folder instead of default continue
  --keep-non-unique # Use all mapped sRNAs in each contig instead of default uniquely mapped sRNAs
  --pearson # Use Pearson correlation instead of default Spearman correlation

For more information, please enter this command in a supported platform such as Ubuntu: bash unisRNA_vfv.sh -h
